# Call Center Performance Dashboard | Power BI

## Project Overview

This project is an interactive Power BI dashboard designed to analyze call center operations, employee productivity, and team performance. The dashboard helps monitor key performance indicators (KPIs), identify trends, and support data-driven decision-making.

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features

- Executive Summary Dashboard
- Department-wise Performance Analysis
- Team Performance Monitoring
- Weekly Performance Tracking
- Call Trend Analysis
- Interactive Filters and Slicers
